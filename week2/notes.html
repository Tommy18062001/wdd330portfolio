W02: Readings and Notes

Chapter 2: Programming Basics

  • JavaScript is a series of statement. And each one ends with a semi-colon
  • A block is a series of statement collected inside curly braces. No semi-colon at the end
  • JavaScript interpreters using an ASI (Automatic Semi-colon Insertion) to place a semi-colon at the end of each line for us.
  • Important Note: Because it can cause an error, it’s considered best practice to write each statement on a new line, terminated by a semi-colon
  • JS has six types of value: string, number, Boolean, null, undefined, objects
  • Variable’s name: Camel Case or Underscore principle
  • Trim(): it’s used to remove whitespace from the beginning and end of a string
  • Octal and Hexadecimal numbers:
    Start with 0x, it is considered to be Hexadecimal
    Start with 0, normally it’s Octal
  • Infinity: It’s an error message in JavaScript to confirm that the number is too big to deal with
  • Type Coercion can be confusing. We need to always verify the type of value we are using before doing any operation
  • undefined is the value given to variables that have not been assigned a value.
  • Null in JavaScript can be thought of a placeholder: “there should be an value here, but there isn’t at the moment.”
  • Bitwise operator: Not ( ~ ), And ( & ), Or ( | ), XOR (exclusive OR) ( ^ ), Shift operator ( << >> )

Chapter 3: Arrays, Logic, Loops

Array properties and method reminder

  • Pop: remove the last item
  • Shift: remove the first item
  • Push: add an item to the end
  • Unshift: add an item at the beginning
  • Concat: used to merge an array with one or more arrays
  • Join: Turn an array into a string
  • Slice: It creates a subarray
  • Splice: remove items from an array and replace it with new items
  • indexOf: find if an item is inside an array. Return -1 if that is the case

The use of Ternary Operation (shorthand)

  • Example: condition ? (If true) : (If false)

Chapter 4: Functions

3 ways to declare a function

  • Functions declaration
  • Functions expressions
  • Function() constructor

Scope: It refers to where a variable is visible and accessible

  • Global scope (any variable declared outside a function has this scope)
  • Local scope (declared inside function using the var statement)

Callback function: it’s a function that is passed as an argument to another function

Array iterators:

  • forEach()
  • map()
  • reduce()
  • filter()
