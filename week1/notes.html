W01: Readings and Notes

Notes

  • The course will provide the skills required to create a web application that run on a mobile device using HTML, CSS, JavaScript
  • Mobile is the new Desktop
  • Responsive is different from Mobile First
  • When writing application, we need to create a bidirectional communication with the user: a way for you to talk to them but as well, a way for the user to speak to you. (Input and output)
  • The section tags fill the screen the way across, it requires the full width.
  • In order to get a user’s input from input tag or text area, use the special word “value” and to show the output, use a section and a special word “innerHTML”.
  • Local Storage use key value pairs.
  • Local Storage will help us store value even though the user closes the page.
